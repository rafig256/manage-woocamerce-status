# مدیریت وضعیت ووکامرس (Manage WooCommerce Status)

این افزونه به شما امکان مدیریت وضعیت‌های سفارشی ووکامرس را می‌دهد.  
نسخه فعلی فقط یک وضعیت پیش‌فرض «چاپ شده» اضافه می‌کند، قبل از وضعیت «تکمیل شد».

## ویژگی‌ها

- افزودن وضعیت پیش‌فرض «چاپ شده»  
- رنگ نارنجی در جدول سفارش‌ها  
- دکمه تغییر وضعیت در پنل مدیریت  
- آماده توسعه برای اضافه کردن، ویرایش و حذف وضعیت‌ها

## نصب

1. پوشه `manage-woocommerce-status` را در مسیر `wp-content/plugins/` آپلود کنید  
2. افزونه را از پیشخوان وردپرس → افزونه‌ها فعال کنید

## استفاده

بعد از فعال‌سازی، وضعیت «چاپ شده» قبل از «تکمیل شد» ظاهر می‌شود و می‌توانید سفارش‌ها را دستی به این وضعیت منتقل کنید.

## سازگاری

- وردپرس 5.0 به بالا  
- WooCommerce 8.x به بالا  
- PHP 7.4 یا بالاتر
